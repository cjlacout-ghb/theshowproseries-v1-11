// turbo-all
1. Check if the application is already running on port 3000 or 3001. If it is, identify the correct URL.
2. If the server is not running, execute `npm.cmd run dev` from the project root.
3. Wait for the terminal to show the "Ready in..." or "Local: http://localhost:..." message.
4. Launch the browser subagent to navigate to the application URL (defaulting to `http://localhost:3000`).
5. Verify that the "The Show Pro Series" dashboard is visible and capture a screenshot.
